# 产品运营团队管理面板 快速上手指南

> 基于本地 JSON + 可选 Notion 同步的项目管理面板。默认使用本地文件存储，无需登录 Notion 即可直接编辑。

## 一、首页五大视图

打开 `http://localhost:3000`（或部署后的域名），顶部 Tab 可切换：

| 视图 | 用途 | 怎么用 |
|------|------|--------|
| **项目看板** | 项目全貌、完成度、风险节点、最新周报进度 | 一眼看整体进度，红色条目优先处理 |
| **决策视图** | 为 Leader 定位卡点、风险、下一步 | 按风险矩阵排序，优先看「高风险」卡片 |
| **节点列表** | 层级化查看所有任务节点 | 点击箭头展开子节点；点击行进入编辑 |
| **任务看板** | 四列拖拽看板 | 拖拽卡片变更状态；卡片展示周报摘要 |
| **甘特图** | 时间线 + 依赖关系 | 点击条形进入节点详情；自动检测循环依赖 |
| **本周汇总** | 按作者查看本周周报 | 展开查看摘要，点击进入编辑 |

## 二、节点（任务）怎么维护

1. **新建节点**：当前需在 Notion 数据库「项目节点」中直接添加（后续可在此面板新增）。
2. **编辑节点**：在列表/看板/甘特图中点击节点名称，进入详情页。
   - 名称、状态、优先级、负责人、起止时间、交付物描述：修改后**失焦自动保存**。
   - 依赖：「依赖于」影响甘特图箭头；「父节点」影响列表层级。
   - 周报：节点详情页下方可直接「写本周周报」或查看历史周报。
3. **字段含义**：
   - `节点状态`：未开始 / 进行中 / 已完成 / 阻塞
   - `优先级`：高 / 中 / 低
   - `起止时间`：用于甘特图与逾期/到期提醒
   - `依赖于`：前置任务；前置未完成时决策视图会提示
   - `父节点`：用于列表层级；子节点会自动挂在父节点下

## 三、周报怎么写（重点）

### 3.1 新建周报

1. 在节点详情页点击「写本周周报」，或在首页点击节点进入后新建。
2. 填写：
   - **作者**：你的名字
   - **周次**：默认本周一，可修改
   - **关联节点**：按住 Ctrl/Cmd 多选；关联后摘要会显示在项目面板
   - **一句话总结**：最关键的信息，会出现在看板/列表中
   - **正文**：支持段落、标题、列表、引用、分隔线、图片、视频
3. 点击「保存」。

### 3.2 添加图片/视频（最省手）

**方法一：拖拽**
- 直接从桌面选中一张或多张图片/视频，拖进正文编辑区，松手即自动上传。

**方法二：粘贴**
- 截图或复制图片后，在正文区按 `Ctrl+V`（Mac `Cmd+V`），自动上传。

**方法三：工具栏**
- 点击工具栏的图片 / 视频图标，选择文件（支持多选）。

> 上传使用 Cloudinary 免费 preset，支持常见图片格式与视频格式。上传成功后会在正文中显示，并随周报一起保存到本地文件（Notion 可用时尝试同步）。

### 3.3 编辑已有周报

- 在首页「本周汇总」中点击周报，或在节点详情页点击历史周报。
- 进入后可修改属性、正文、增删图片视频。
- 修改后点击右上角「保存」。

### 3.4 周报模板示例

新建周报时，正文已预填示例，可直接修改：

```
# 本周核心进展
完成了 XX 模块的原型设计，已与产品侧确认并进入开发排期。

## 关键数据/成果
• 用户访谈 5 人，整理核心痛点 8 条
• 输出竞品分析报告 1 份

## 下周计划
1. 完成需求评审并拆分子任务
2. 启动 UI 设计初稿

## 风险与求助
> 目前无阻塞风险；如需协调开发资源请 @ 相关负责人。
```

## 四、数据联动说明

- **节点 ←→ 周报**：周报关联节点后，节点卡片/列表会显示该节点**最新一条**周报的一句话总结。
- **周报 ←→ 面板**：周报中关联的节点名称后会显示节点实时状态，方便写周报时对照。
- **状态 ←→ 进度**：项目看板中的完成度按「已完成=100%、进行中=50%、未开始/阻塞=0%」计算。
- **风险 ←→ 优先级+时间**：决策视图的风险评分综合了优先级、逾期、阻塞、即将到期等因素。

## 五、常见问题

**Q：上传图片/视频失败怎么办？**
A：检查 `.env.local` 中 `NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME` 和 `NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET` 是否配置正确，以及 Cloudinary unsigned upload preset 是否开启。

**Q：周报保存后正文格式变了？**
A：Notion 会按自己的规则渲染部分块。复杂格式建议在 Notion 中二次微调；基础段落、标题、列表、图片、视频可正常同步。

**Q：决策视图里的风险是怎么算的？**
A：高优先级 + 阻塞/逾期 = 高风险；阻塞或逾期 = 中风险；3 天内到期 = 中/低风险。具体见 `lib/analytics.ts`。

**Q：能在看板里直接改状态吗？**
A：可以。拖拽卡片到「进行中/已完成/阻塞/未开始」列即可自动保存。
